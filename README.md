# nextJS-assignment 
- 노마드코더 Next.js - 과제용

## Setting
1. npm init
```
npm init -y
```
2. package.json script 명령어, license 수정 
```
  "scripts": {
    "dev": "next dev"
  }, 
  "license": "MIT"
```
3. react,react-dom, next 설치
```
npm install react@latest next@latest react-dom@latest
```
4. Script
```
dev :  npm run dev 
build : npm run build 
```


## Tech Stack
- Next.js
- TypeScript 
- React Server Components
- CSS Modules
